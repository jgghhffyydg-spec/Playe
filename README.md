# Playe

Playe is a Python/Kivy application for learning English words and sentences with organized daily repetition.

## Goal

The app helps learners at all levels add English phrases, repeat each new phrase 10 times immediately, then review it again after 10 minutes, 1 hour, 1 day, and 1 week.

## Run

```bash
python -m pip install kivy
python main.py
```

Review data is stored locally in `phrases.json`.
